# A/B test comparison

## Overview

Comparing frequentist and bayesian A/B testing methods using an insurance claim form experiment as a practical case study. The project evaluates two front-end UX designs - one asking for a cost estimate in a generic way and the other relying on an accuracy nudge - to observe how nuding affects the accuracy of the estimates.

The simulated experiment serves as a concrete example to demonstrate how frequentist and bayesian A/B testing paradigms interpret identical behavioral data.
